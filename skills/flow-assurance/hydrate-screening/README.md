# Hydrate Screening

Educational hydrate-risk screening placeholder for public NeqSim community examples.

The screener uses pressure, temperature, and water presence to produce a simple risk level and margin indicator. Real hydrate predictions should use validated NeqSim methods.

## Install

```bash
python -m pip install -e skills/flow-assurance/hydrate-screening
```

## Run Example

```bash
python skills/flow-assurance/hydrate-screening/examples/simple_hydrate_screening.py
```

## Run Tests

```bash
python -m pytest skills/flow-assurance/hydrate-screening/tests
```

## Public Scope

This skill does not include proprietary hydrate curves, confidential operating envelopes, inhibitor strategies, or company-specific standards.
