# Skill Template

Copy this folder when adding a new public community skill.

## Required Steps

1. Rename the folder using kebab-case.
2. Rename the catalog skill in `SKILL.md` so it starts with `neqsim-`.
3. Replace all placeholder text.
4. Add Python code under `src` if needed.
5. Add at least one public example under `examples`.
6. Add pytest tests under `tests`.
7. Run the repository test suite.

Do not add confidential data, company-specific methods, private URLs, or unpublished standards.
